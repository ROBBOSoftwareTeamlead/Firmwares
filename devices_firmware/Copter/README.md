# Прошивка квадрокоптера (STM32)

**cf2-2023.07.bin** — прошивка для основного процессора Crazyflie 2.x (STM32). Источник: [Bitcraze crazyflie-firmware](https://github.com/bitcraze/crazyflie-firmware), релиз 2023.07.

Радиомодуль на борту (NRF51) прошивается отдельно: [crazyflie2-nrf-firmware](https://github.com/bitcraze/crazyflie2-nrf-firmware), релиз 2023.02 (cf2_nrf-2023.02.bin). Для Robbo Scratch используется связка Crazyradio (донгл) + эта прошивка STM32.
